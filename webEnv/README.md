# WebEnv

#### 介绍
爬虫补环境工具，拿到环境后先npm install安装环境依赖，使用时如报错NodePlugin\build\Release\faker.node的错误，
是由于document.all环境引起，需安装该文档：https://blog.csdn.net/randy521520/article/details/145338413
重新构建NodePlugin，该工具中的NodePlugin基于node v23.11.0版本构建，如不像费事构建可使用node v23.11.0版本
工具教程：https://blog.csdn.net/randy521520/article/details/147014083
